# todo — タスク管理ツール

ブラウザだけで動くシンプルなタスク管理アプリです。ビルドや依存関係は不要で、`index.html` を開くだけで使えます。

## 機能

### 長期目標(TOEIC600点なども)

- 🎯 **目標カード** — 目標名・カテゴリ・目標日を入れると「やることリスト」を自動生成(試験・資格 / 学習 / 運動 / 読書 / 貯金 / 汎用)
- ▶ **次の一歩だけ表示** — 未完了ステップの先頭だけを提示。ワンクリックで今日のタスクに追加(ステップの締め切りも引き継ぐ)
- 📅 **ステップごとの締め切り** — 各ステップに期限を設定でき、締め切りレーダーで監視。「残りステップに締切を自動で割り振る」で目標日から均等に自動設定
- 🔗 **タスクと連動** — 追加したタスクを完了するとステップも自動完了、XP・ストリークも貯まる
- 📈 **逆算ペース表示** — 「あと◯日・週◯ステップでOK」
- 🏆 **全ステップ達成で盛大にお祝い**

### 継続をサポートする仕掛け

- 🔥 **連続記録(ストリーク)** — 1日1件でも完了すれば記録が伸びる
- ⭐ **XPとレベル** — タスクを完了するたびに経験値が貯まってレベルアップ
- 🎯 **今日の目標** — 「まず3件」の小さなゴールに集中できる進捗バー
- 🔁 **習慣タスク** — 「毎日繰り返す」にチェックすると毎日リセットされるタスクに
- 🎉 **完了時のごほうび** — 紙吹雪と褒め言葉で即時フィードバック
- 📅 **直近7日の活動ドット** — 頑張りがひと目でわかる

### 教材の進捗管理

- 📚 **教材を登録** — 問題集・参考書・動画講座など(全体量と単位: ページ/章/問/レッスン/回)
- 🔄 **周回(復習)前提** — デフォルト3周。1周終わると自動で次の周へ。現在の周と全体進捗%を表示
- 🧠 **復習ペースの自動配分** — 復習は速く回せる前提(1周目:2周目:3周目 = 1:0.6:0.4)で、目標日までの日数を各周に配分し「1周目は8/10までに(1日◯ページ)→ 2周目→9/1 → 3周目→9/15」と締切目安を逆算
- ➕ **ワンタップ更新** — +1/+5/+10 ボタンか「ここまで進んだ」入力で進捗バーが伸びる
- 🧮 **実績ペース** — 直近7日の進み具合から今の周の完了予測
- 進捗更新もXP・ストリークにカウント(1教材につき1日1回)、周回完走ごとにお祝い、全周完走で盛大にお祝い

### 締め切りレーダー

- ⏰ **一括監視** — タスクの期限・目標の目標日・教材の目標日をまとめて一覧表示
- 🚨 **緊急度で色分け** — 「◯日超過」「今日!」「明日」「あと◯日」
- 🔔 **開いた瞬間に警告** — 超過・今日締切があればトースト表示+タブタイトルに件数バッジ

### 学びメモ

- 📝 **クイックメモ** — 今日学んだこと・思ったことを1行から記録(Ctrl+Enter対応)。書くだけでXP・ストリークにカウント
- #️⃣ **ハッシュタグ** — 本文に `#英語` のように書くだけでタグ化。チップをクリックして絞り込み
- 📅 **日付ごとの自動整理** — 「今日」「昨日」「◯月◯日(曜)」で自動グループ化
- 🔍 **全文検索** / ⏪ **ランダム振り返り** — 過去のメモをランダムに再表示

### バックアップと同期

- 📤 **書き出し / 📥 取り込み** — 全データを文字列コピーまたはJSONファイルで書き出し、別の端末で取り込み(バックアップ兼用)
- 🔄 **自動同期(GitHub Gist)** — gist権限のトークンを各端末に貼るだけで、非公開Gist経由でスマホ⇔PCが自動同期(新しい方が勝つ方式)。ページを開いたとき・変更したとき・タブに戻ったときに同期
- ※データは通常、端末ごとのブラウザ(localStorage)に保存されます

### 基本機能

- タスクの追加(タイトル・期限・優先度)
- 完了/未完了の切り替え
- タイトルの編集(✏️ ボタン、またはタイトルをダブルクリック)
- タスクの削除・完了済みの一括削除
- フィルター(すべて / 未完了 / 完了)
- キーワード検索
- 並び替え(作成順 / 期限順 / 優先度順)
- 期限切れタスクのハイライト
- localStorage による自動保存(ブラウザに保存されます)
- ダークモード対応(OS の設定に追従)

## 使い方

```bash
# そのままブラウザで開く
open index.html

# または簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000 を開く
```
