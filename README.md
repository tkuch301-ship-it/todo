# todo — タスク管理ツール

ブラウザだけで動くシンプルなタスク管理アプリです。ビルドや依存関係は不要で、`index.html` を開くだけで使えます。

## 機能

### 継続をサポートする仕掛け

- 🔥 **連続記録(ストリーク)** — 1日1件でも完了すれば記録が伸びる
- ⭐ **XPとレベル** — タスクを完了するたびに経験値が貯まってレベルアップ
- 🎯 **今日の目標** — 「まず3件」の小さなゴールに集中できる進捗バー
- 🔁 **習慣タスク** — 「毎日繰り返す」にチェックすると毎日リセットされるタスクに
- 🎉 **完了時のごほうび** — 紙吹雪と褒め言葉で即時フィードバック
- 📅 **直近7日の活動ドット** — 頑張りがひと目でわかる

### 基本機能

- タスクの追加(タイトル・期限・優先度)
- 完了/未完了の切り替え
- タイトルの編集(✏️ ボタン、またはタイトルをダブルクリック)
- タスクの削除・完了済みの一括削除
- フィルター(すべて / 未完了 / 完了)
- キーワード検索
- 並び替え(作成順 / 期限順 / 優先度順)
- 期限切れタスクのハイライト
- localStorage による自動保存(ブラウザに保存されます)
- ダークモード対応(OS の設定に追従)

## 使い方

```bash
# そのままブラウザで開く
open index.html

# または簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 構成

```
index.html    … 画面
css/style.css … スタイル
js/app.js     … ロジック(依存関係なし)
```
