# todo — タスク管理ツール

ブラウザだけで動くシンプルなタスク管理アプリです。ビルドや依存関係は不要で、`index.html` を開くだけで使えます。

## 機能

- タスクの追加(タイトル・期限・優先度)
- 完了/未完了の切り替え
- タイトルの編集(✏️ ボタン、またはタイトルをダブルクリック)
- タスクの削除・完了済みの一括削除
- フィルター(すべて / 未完了 / 完了)
- キーワード検索
- 並び替え(作成順 / 期限順 / 優先度順)
- 期限切れタスクのハイライト
- localStorage による自動保存(ブラウザに保存されます)
- ダークモード対応(OS の設定に追従)

## 使い方

```bash
# そのままブラウザで開く
open index.html

# または簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## 構成

```
index.html    … 画面
css/style.css … スタイル
js/app.js     … ロジック(依存関係なし)
```
